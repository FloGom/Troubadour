# Volume script install

## Install
1. Copy the file *vol* in [Troubadour/frontPanelProg](../frontPanelProg) in `/usr/local/bin/`
2. Make it executable :   
`sudo chmod a+x /usr/local/bin/vol`
3. Test it by typing :   
`vol` in a terminal, this command outputs the current volume as a number between 0 and 100
